# Psych Quiz

简短心理健康自查小测验：测压力、查睡眠、看状态，结果仅供自我观察参考

## What it does
- 提供压力、睡眠、情绪过载等主题的 5-7 题简短自评问卷
- 根据 Never/Sometimes/Often/Almost always 四级评分，输出结果分档与解读
- 给出 1-3 条实用建议，并在适当场景提醒寻求专业支持

## Example scenarios

**1. 压力自测**
> 👤 帮我测测压力，看我现在的压力有多大
> 🤖 6道题：紧张感、思绪纷乱、易怒、睡眠质量、负荷感知、注意力——每题Never/Sometimes/Often/Almost always四级。计分后输出结果分档（可控/累积/升高/较高），简短解读加2-3条建议

**2. 睡眠质量自查**
> 👤 最近睡得不好，帮我看看是不是太焦虑了
> 🤖 5道睡眠题：入睡、保持睡眠、醒后疲惫、日间倦意、睡前思绪。评分后判断睡眠问题是压力驱动还是其他原因，给1-2个下一步（如睡前放松流程、减少睡前屏幕时间）

**3. 网上的心理测试能信吗？**
> 👤 朋友分享了一个公众号的心理健康测试说很准，做完显示中度抑郁，我该信吗？
> 🤖 提醒：公众号非专业测试不具备临床诊断效度。建议使用标准化工具（SDS抑郁自评量表、GAD-7焦虑筛查量表，简单心理/壹心理App有官方版本）。如非专业测试提示中度以上，建议去三甲医院心理科或精神科做专业评估。避免反复做不同在线测试造成"诊断色"
